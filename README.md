# 6G-RIS-DESIGN
The algorithm comparison is implemented by ResultPSO, ResultIPM, and PSOIPM.
The RIS model file is ThreeModel2.cst.
Simulated S-parameter data are provided.
Algorithm iteration data are obtained from PSO.fig, IPM.fig, and x0 instantiated IPM.fig.
For plotting, the following codes can be used: DieDaiTu, XiangWei, XiangWeiDuiBi, and SE.
Parameter mapping: A0 corresponds to parameter a in the paper, C0 corresponds to b, B0 corresponds to c, and D0 corresponds to d.
Ensure compatibility with CST 2022 and MATLAB 2023a (or newer versions).
